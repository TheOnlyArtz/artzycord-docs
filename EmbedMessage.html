EmbedMessage

A class that allows the client to send Embed objects

new EmbedMessage()

Source:
  • EmbedMessage.js, line 20
Properties:
Name Type Description
fields Array The array of the embed fields
color Number The color of the embed (RGB color codes ONLY)
description String The embed"s description
title String The embed's title
footer String The embed's footer
author String The embed's author
provider String The embed's provider
image URI The embed's image (URL must be image resolvable)
thumbnail URI The embed"s thumbnail (URL must be image resolvable)
video URI The embed's video (URL mest be video resolvable)
Example
// creating a simple embed
let myEmbed = new Discord.EmbedMessage()
		.newDescription("Hey. this is how to make an embed with ArtzyCord")
		.newThumbnail("https://goo.gl/Sgg1hD")
message.channel.send(myEmbed);
